## 📦 Installation

Make sure you have [Node.js](https://nodejs.org/) installed, then run:

```sh
yarn install
```

## 🏃 Running the Project

To start the development server, run:

```sh
yarn dev
```

Then open [http://localhost:3000/api-docs](http://localhost:3000/api-docs) in your browser.
